# Проект: Парсер вакансий с HH.ru и работа с базой данных PostgreSQL

Данный проект предназначен для сбора данных о компаниях и их вакансиях с сайта hh.ru, сохранения этой информации в базу данных PostgreSQL и удобного управления ею через Python.
Основные функции

    * Сбор данных: Автоматическое получение информации о 10 крупных компаниях и их открытых вакансиях через API HeadHunter.

    * Работа с БД: Автоматическое создание таблиц и заполнение их данными с использованием библиотеки psycopg2.

    * Аналитика: Специальный класс DBManager позволяет выполнять SQL-запросы для анализа вакансий:
    
    * Подсчет количества вакансий у компаний.

    * Вывод всех вакансий с зарплатой.

    * Расчет средней зарплаты.

    * Поиск вакансий по ключевому слову.

# Требования

### Для работы проекта необходимы:

    Python 3.10+

    СУБД PostgreSQL

    Библиотеки: requests, psycopg2-binary

### Установка и запуск

Клонируйте репозиторий или скачайте архив с кодом.

### Установите зависимости:

    pip install -r requirements.txt

### Настройте базу данных:

Создайте в PostgreSQL базу данных (например, hh_course).

В корне проекта создайте файл database.ini на основе следующего шаблона:


>[postgresql]\
host=localhost\
user=postgres\
password=ваш_пароль\
port=5432\
database=hh_course\

### Запустите основной скрипт:


    python main.py

Структура проекта

    main.py — основной файл для взаимодействия с пользователем.

    hh_parser.py — модуль для работы с API HeadHunter.

    db_manager.py — класс с методами для выполнения SQL-запросов к БД.

    utils.py — вспомогательные функции для создания таблиц и сохранения данных.

    config.py — логика чтения конфигурации БД из .ini файла.
